# 💌 [《巷口的热豆浆》](https://findonei.github.io/)

老巷的青石板路被晨雾浸得发潮，阿婆的豆浆摊支在巷口第三棵榕树下，不锈钢桶里的热气裹着豆香，能飘到巷尾的裁缝铺。

阿明每天七点准到，要两碗甜豆浆，加半勺糖。他把其中一碗推到对面，塑料碗沿还沾着细密的水珠。阿珍总系着洗得发白的蓝布围裙，指尖沾着针脚线尾，接过碗时会轻轻碰一下他的手背，像两片落在水面的柳叶。

没人知道他们是从什么时候开始的，只记得阿珍刚租下裁缝铺时，缝纫机的嗒嗒声总盖过阿婆的叫卖。后来阿明会在收摊前帮她把门口的布料搬进屋里，阿珍会把裁剩下的碎花布缝成杯套，套在阿明的搪瓷缸上。

上周阿珍的母亲来城里看病，阿明凌晨三点就去排队挂号，回来时豆浆摊的火已经生好，阿婆把一碗热豆浆塞给他，说“加了双份糖”。那天阿珍没去开门，傍晚时她端着一篮煮好的玉米站在巷口，夕阳把她的影子拉得很长，阿明接过玉米时，发现她的眼眶红着，却笑出了两个梨涡。

昨天我路过巷口，看见阿明蹲在地上帮阿珍修缝纫机，阿珍坐在小板凳上，把剥好的橘子一瓣一瓣放进他嘴里。风穿过榕树的枝叶，把豆浆的香气、橘子的酸甜和缝纫机嗒嗒的预备声，揉成了老巷最温柔的清晨。

金句： 最好的爱情，是把细碎的日子熬成糖，每一口都暖到心坎里。
